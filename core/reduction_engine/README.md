# Reduction Engine

Polynomial-time reductions among canonical NP-complete problems.

Implemented transformations:
- SAT -> 3-SAT
- 3-SAT -> Clique / Independent Set / Vertex Cover / Graph 3-Coloring / Hamiltonian Circuit

Includes verification helpers and `ReductionChain` for composition.

References: Karp (1972), Garey-Johnson (1979).
